# web

An app for displaying movies.


## Installation

Copy the `.env.local` example and complete with required variables:

```bash
cp .env.example .env
```

Run the following command:

```bash
npm install
```


## Processing

Run the following command to start dev server:

```bash
npm run dev
```

Run the following commands to start linting and check types:

```bash
npm run lint:css
```

or

```bash
npm run lint:js
```

and

```bash
pnpm run check-types
```

Run the following command to open a live tunnel:

```bash
npm run tunnel
```
