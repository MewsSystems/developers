# Colours

Utilities for controlling colours. Accessed via the class
`{bg|border|fill|text}-{name}-{id}`.

## Brand colours

These colours are defined in the brand guidelines. Primary colours are most
frequently used across the UI and imparts a distinct identity strengthening
brand awareness. Secondary colours highlight or complement the primary colour
and should be used sparingly to make the UI elements stand out.
