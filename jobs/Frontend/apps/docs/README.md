# docs

A design system for a movie app.


## Installation

Run the following commands:

```bash
pnpm install
```


## Processing

Run the following command to start dev server:

```bash
pnpm run dev
````

Run the following commands to start linting:

```bash
pnpm run lint
```

Run the following command to open a live tunnel:

```bash
npm run tunnel
```
