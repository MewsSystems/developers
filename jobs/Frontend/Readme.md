# Movie Search app

## Install the dependencies

`npm install`

## Run the app locally

`npm run dev`

## Build the app

`npm run build`

## Serve the production build locally

`npm run preview`

## Run the unit tests

`npm run test`

## Check the typing

`npm run checkts`

## About prettier
To automatically format the code: `npm run format`.
Ideally, this should be done on commit.
