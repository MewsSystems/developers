- [Node.js](https://nodejs.org) (v20.12.0 or newer)
- [npm](https://www.npmjs.com) (v10.5.0 or newer)

## Getting Started

### Development

```shellscript
# Start the Vite dev server
npm run dev 
```

### Deployment

```shellscript
# Build the app for production
npm run build

# Start the app in production mode
npm start
```
