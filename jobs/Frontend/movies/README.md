# Movies app

## Configure 

Set env variables or use `.env` files for configuration. See `.env.sample` for a reference.

```sh
npm install
```

## Dev

```sh
npm run dev
```

## Test

```sh
npm run test
```

## Build

```sh
npm run build
```

[Based on vite react template](README.VITE.md)
