# ui

A package for the UI library.


## Installation

Run the following commands:

```bash
pnpm install
```


## Processing

Run the following command to build assets:

```bash
pnpm run dev
````

Run the following commands to start linting and check types:

```bash
pnpm run lint
```

and

```bash
pnpm run check-types
```
